Welcome to the certportal

You can use this portal to request a NLX development certificate.

1. Generate a certificate request

First generate a certificate request and a private local key with the following command: 

openssl req -utf8 -nodes -sha256 -newkey rsa:2048 -keyout {yourhostname}.key -out {yourhostname}.csr

2. Enter certificate request details

Now enter the contents of {yourhostname}.csr in the following field:
